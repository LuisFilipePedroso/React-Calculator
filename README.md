## The Project

A simple project that can be useful for you to calculate simple math operations

## Tech Stack

ReactJS
NextJS
Typescript
ChrakraUI

## Running the Project

First, you must install the dependencies:

`yarn`

Then, to run the project, execute:

`yarn dev`
